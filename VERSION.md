# 版本说明

- **版本号**：v1.0-baseline
- **冻结日期**：2026-06-26
- **状态**：✅ 已冻结，可作为后续优化的稳定基线

## 本版主要改动

### 1. 时间轴节点上显示个股信息
- 每个时间轴节点现在展示当日有活动的最多 3 个个股（带操作图标 + ticker）
- 超出显示 +N 提示
- 鼠标悬停的 tooltip 包含：个股名 / ticker / 操作类型 / 快照价格 / 所属板块与细分

### 2. 板块编辑能力增强
- 板块表单新增 **图标 (emoji + 选择器)** 字段
- 板块表单新增 **主题色** 字段
- 板块表单新增 **板块级跟踪管理** 区域（列出 / 编辑 / 删除 / 新增整体跟踪）
- 主页 / 项目视图 / 板块详情页都显示板块图标
- 修复了之前细分领域 edit-tracking 按钮（点击无反应）

### 3. 修复细分领域图标不能选择
- 根因：.form-row 是两列 grid，但 label + input + picker 有 3 个子元素，picker 被挤到下一行
- 修复：用 .form-cell 包裹 input + picker，恢复成两列布局
- 同样修复应用到 ormChainNode

### 4. 中文字体渲染修复
- --font 字体栈：中文字体前置（PingFang SC / Microsoft YaHei / 微软雅黑 / Noto Sans CJK SC 等）
- --mono 字体栈：西文等宽字体后追加中文字体回退
- 覆盖 Windows / macOS / Linux 三平台

## 已知不足（可优化方向）

- 板块级跟踪目前没有「快速操作」按钮（用全功能表单）
- 板块图标选择范围只有 32 个预设 emoji，可考虑支持自定义文本
- 主页时间轴节点的个股胶囊超过 3 个时只显示 +N，无法展开看全部
- 时间轴 / 主页之间的视图切换没有动画
- 板块编辑表单的跟踪管理区域滚动条样式可以更精细
